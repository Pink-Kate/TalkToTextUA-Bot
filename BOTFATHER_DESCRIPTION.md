# Опис для BotFather

## Name
TalkToText UA

## Short Description (≤120 символів)
Миттєвий перетворювач голосових у текст. Українська + авто-визначення мови.

## Description
Бот приймає голосові/аудіо та повертає чистий текст. Працює з українською, англійською, польською та ін. Автовизначення мови, експортує .txt за запитом, підтримує довгі повідомлення.

## Commands

### /start
Привітання й інструкція

### /help
Підказки та приклади

### /lang
Виставити мову розпізнавання

**Використання:**
- `/lang uk` — українська
- `/lang en` — англійська
- `/lang pl` — польська
- `/lang auto` — авто-визначення

### /mode
Точність/швидкість

**Використання:**
- `/mode fast` — швидкий режим (менша точність)
- `/mode balanced` — збалансований режим (за замовчуванням)
- `/mode accurate` — точний режим (повільніше, але точніше)

### /export
Надіслати .txt файлом

Експортує всю історію розпізнавань у текстовому файлі.

### /clear
Стерти історію (тільки метадані)

Очищає історію діалогу та метадані користувача.

### /privacy
Приватність та згода

Показує інформацію про обробку даних та приватність.

## Як налаштувати в BotFather

1. Відкрийте @BotFather в Telegram
2. Виберіть `/mybots`
3. Виберіть вашого бота
4. Виберіть "Edit Bot"
5. Виберіть "Edit Description" та вставте опис вище
6. Виберіть "Edit About" та вставте короткий опис
7. Виберіть "Edit Commands" та додайте команди:
   - start - Привітання й інструкція
   - help - Підказки та приклади
   - lang - Виставити мову розпізнавання
   - mode - Точність/швидкість
   - export - Надіслати .txt файлом
   - clear - Стерти історію
   - privacy - Приватність та згода

## Налаштування для роботи в групах

Для того, щоб бот працював у групах:

1. Відкрийте @BotFather
2. Виберіть вашого бота
3. Виберіть "Bot Settings"
4. Виберіть "Group Privacy"
5. Встановіть "Disable" (вимкніть приватність групи)

Після цього бот буде отримувати всі повідомлення в групах, включаючи голосові.
